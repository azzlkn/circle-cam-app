# Circle Cam App

A desktop webcam application for streamers.

![CircleCamApp](./preview.png)

### Install

To run the app:

```
npm install
npm run start
```

To build the app:

```
npm run app:dir
npm run app:dist
```

**The app was only tested on Mac M1.**
